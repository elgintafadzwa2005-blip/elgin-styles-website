# Elgin Styles — Website

Simple static site for Elgin Styles built with semantic HTML and separated CSS. Includes a contact form wired to Formspree (replace the placeholder form ID with your Formspree form's ID).

Features
- Semantic, accessible HTML structure
- Separated stylesheet (styles.css)
- Responsive layout for mobile and desktop
- Contact form with Formspree placeholder (no server required)

Usage
1. Replace the Formspree action URL in `index.html` with your Formspree form endpoint, for example:

   action="https://formspree.io/f/mayxyzab"

2. Serve the files from any static host (GitHub Pages, Netlify, Vercel, etc.) or open `index.html` locally.

3. Update content, images, and contact details to match your salon.
